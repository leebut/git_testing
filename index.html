CSS Flexbox
  • display: flex;
  • flex-direction:
  • justify-content:
  • align-items:
  • flex-wrap:
display: flex;

display: flex; instructs the browser to convert a div element into a flex.
Look at the following example:

CSS code:

.container {
  display: flex;
}

HTML code:

<div class="container">
  This is a flex container
</div>
flex-direction:

The flex direction determines the direction of the main and cross axes.

This can be a little confusing, as when the desired result is two columns, you will set the flex-direction to row so that the element can be divided horizonatally. Conversely, setting flex-direction: column; will set a vertical main axis, and items will be added along it.
The following code creates a 50-50 horizontal split of the flex container.

CSS code:

.container {
  display: flex;
  flex-direction: row;
  width: 100vw;
}
.col {
  width: 50%;
}
.left-col {
  background: yellow;
}
.right-col {
  background: blue;
}


HTML code:

<div class="container">
  <div class="col left-col">
    Left column
  </div>
  <div class="col right-col">
    Right column
  </div>
</div>
justify-content:

justify-content: aligns flex items along the main axis. The items can be aligned using the following attributes:

  • flex-start - aligns items to the left of the container
  • center - centers the items horizonatally
  • flex-end - aligns the items to the right of the container
  • space-between - calculates space between each flex item and pushes the outermost items to the container's edges.
  • space-around - similar to space between, but also adds space between the outermost items and the edges.
  • self - allows alignment of a single element
align-items:

align-items: aligns flex items along the cross axis. The items can be aligned using the following attributes:

  • flex-start - aligns items to the top of the container
  • center - centers the items vertically
  • flex-end - aligns the items to the bottom of the container
  • space-between - calculates space between each flex item and pushes the outermost items to the container's vertical edges.
  • space-around - similar to space between, but also adds space between the outermost items and the vertical edges.
  • self - allows alignment of a single element
flex-wrap:

flex-wrap: forces flex items to wrap onto a new row if there is no more room in the viewport. This property can be used is some cases to avoid the use of breakpoints.

Without flex-wrap: wrap; the flex items would continue to be squashed as the viewport size shrinks.
